Problema

Planteamiento del Problema

Surgimiento de la idea

La poca gestión en el manejo de las contraseñas y datos sensibles reposados en las diferentes plataformas de información resaltan la gran vulnerabilidad que poseen los datos si no se protegen para lo cual se precisa de una herramienta móvil que ayude en la gestión de tales datos sensibles. El proyecto inicialmente se pensó a fin de cubrir una necesidad de propia del equipo desarrollador que al igual siente la necesidad y busca cubrirla, actualmente el uso de aplicativos se hace más frecuente en nuestras vidas e ingresar los datos personales en ellos es propio del día a día lo que hace imperativo el ejercer seguridad en todo contexto.
Noviembre 2022

Pregunta de Investigación

Interrogante

¿Cómo mitigar la falta de gestión en la administración y almacenamiento de datos sensibles y contraseñas, requeridos en las diferentes plataformas y sistemas de información para la autenticación de usuario, haciendo uso de una herramienta tecnológica de acceso confiable y seguro?

Objetivos



Surgimiento de la idea

Objetivo general


Desarrollar una aplicación móvil que permita centralizar y administrar la gestión de contraseñas de forma segura y responsable permitiendo al usuario una reducción en la repetición de estas, requiriendo únicamente de una sola clave de ingreso.



Objetivos especificos




► Maquetar la base de datos de la aplicación.

► Diseñar las interfaces gráficas que permitan el uso fluido de la aplicación

► Desarrollar un aplicativo web que ejecute las mismas funcionalidades que la aplicación móvil



El abordaje a los objetivos planteados apuntan a desarrollo en dispositivos de mayor uso en el páis

Los equipos celulares son los dispositivos más usados en Colombia, seguido de los equipos portatiles y/o computadores de escritorio

DROP PROCEDURE IF EXISTS busqueda; #terminado // BUSQUEDA
DELIMITER //
CREATE PROCEDURE busqueda(IN sitioBuscado VARCHAR(45),IN idusuario INT)
BEGIN
   SELECT id_sitio, nombre_s, usuario_s, email_s, password_s, fechacreado from Sitios WHERE id_usuario_s = idusuario AND nombre_s LIKE (SELECT CONCAT('%',sitioBuscado,'%'));
END //

#Se crea procedimiento Almacenado
DROP PROCEDURE IF EXISTS guardarSitio; #terminado // NUEVO SITIO
DELIMITER //
CREATE PROCEDURE guardarSitio(IN nombreIn VARCHAR(45),IN usuarioIn VARCHAR(45),IN emailIn VARCHAR(255),IN passwordIn VARCHAR(60),IN idusuarioIN INT)
BEGIN
   INSERT INTO Sitios (nombre_s, usuario_s, email_s, password_s, id_usuario) VALUES (nombreIn, usuarioIn, emailIn, codificado(passwordIn), idusuarioIN);
END //


#Se crea procedimiento Almacenado
DROP PROCEDURE IF EXISTS consultargeneral; #terminado // CONSULTAR DE FORMA GENERAL
DELIMITER //
CREATE PROCEDURE consultargeneral(IN idusuario INT(10))
BEGIN
   SELECT id_sitio, nombre_s, usuario_s, email_s, password_s, fechacreado FROM Sitios WHERE id_usuario_s = idusuario;
END //


#Se crea procedimiento Almacenado
DROP PROCEDURE IF EXISTS editar; #terminado // ACTUALIZAR
DELIMITER //
CREATE PROCEDURE editar(IN nombreIn VARCHAR(45),IN usuarioIn VARCHAR(45),IN emailIn VARCHAR(255),IN passwordIn VARCHAR(60),IN idIn INT,IN idSitioIn INT)
BEGIN
   UPDATE Sitios SET nombre_s = nombreIn, usuario_s = usuarioIn, email_s = emailIn, password_s = codificado(passwordIn) WHERE id_sitio = idIn AND id_usuario_s = idSitioIn;
END //

#Se crea procedimiento Almacenado
DROP PROCEDURE IF EXISTS eliminar; #terminado // ELIMINAR
DELIMITER //
CREATE PROCEDURE eliminar(IN idIn INT)
BEGIN
   DELETE FROM Sitios WHERE id_sitio = idIn;
END //


#Se crea procedimiento Almacenado
DROP PROCEDURE IF EXISTS iniciarsesion; #terminado // ELIMINAR
DELIMITER //
CREATE PROCEDURE iniciarsesion(IN usuario VARCHAR(60),IN contra VARCHAR(60))
BEGIN
      SELECT id_usuario, usuario_u, llavemaestra FROM Usuarios WHERE Usuarios.usuario_u = usuario AND Usuarios.llavemaestra = codificado(contra);
END //




#Se crea TRIGGER 
DROP TRIGGER IF EXISTS almacenar; #terminado // ALMACENAR CONTRASEÑAS VIEJAS
DELIMITER //
CREATE TRIGGER almacenar BEFORE UPDATE ON Sitios
FOR EACH ROW
BEGIN
   INSERT INTO Repositorios (nombreSitio,usuarioSitio,emailSitio,passwordSitio,fecha,comentario)
   values (OLD.nombre_s, OLD.usuario_s,OLD.email_s,OLD.password_s,now(),"Sitio Actualizado");
END //
   

#Se crea funcion de codificado
DROP FUNCTION IF EXISTS codificado;
DELIMITER //
CREATE FUNCTION codificado(pass VARCHAR (60))
   RETURNS VARCHAR(255) deterministisc
   BEGIN
   DECLARE a VARCHAR(255);
   SET a = hex(aes_encrypt(pass,'password'));
   RETURN a;
END //


#Se crea funcion de Decodificado
DROP FUNCTION IF EXISTS decodificado;
DELIMITER //
CREATE FUNCTION decodificado(pass VARCHAR(255))
   RETURNS VARCHAR (60) deterministisc
   BEGIN
   DECLARE a VARCHAR(60);
   SET a = aes_decrypt(unhex(pass),'password');

   RETURN a;
END //
